## 📌 Project Description
This project is a **TaskFlow Pro - Advanced To-Do List Application** built as Task 04 for SkillCraft Technology Internship. 
It helps users manage tasks across multiple lists with priorities, due dates, search, and progress tracking.
## 🎯 Objective
To create a feature-rich, responsive task management app that helps users organize, track, and complete their daily tasks efficiently.
## ✨ Features Implemented
- **Multiple Lists**: Create, switch and delete custom task lists like Work, Personal
- **Add/Edit/Delete Tasks**: Full CRUD operations for tasks
- **Priority Levels**: Set tasks as High, Medium, Low with color-coded indicators
- **Due Date & Time**: Add deadlines with automatic overdue detection and alerts
- **Search & Sort**: Search tasks and sort by Date, Priority, or Name
- **Progress Ring**: Visual completion percentage for the active list
- **Stats Dashboard**: Track Total, Active, Completed, and Overdue tasks
- **LocalStorage**: All data saves automatically in browser
- **Edit Task Modal**: Update task details without deleting
- **Dark/Light Theme Toggle**: Switch themes for comfort
- **Responsive Design**: Works perfectly on desktop and mobile
## 🛠️ Technologies Used
- **HTML5** - Semantic structure
- **CSS3** - Styling, Grid, Flexbox, Glassmorphism, Animations
- **JavaScript (ES6)** - DOM manipulation, LocalStorage, Date handling
